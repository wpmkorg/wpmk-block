== Description ==

<h3>WPMK Block</h3>
This <strong>WPMK Block</strong> Plugin helps you enhance your website design and functionality, and you can also create many design with help of this plugin, you do not need to do any coding just add this plugin in your runing website and use / play with your design and functionality. This plugin will support all the plugins, theme and also will work with visual composer.

= Plugin Features =

1. Tinymce Shortcode.
2. Visual Composer Shortcode.
3. Block Widget.
4. Custom Post Type.
5. Light Wight.
6. Support all plugins and themes.
7. We are working on Page Builder and Elementor.

= Plugin ideal usage =

1. Enhance Your website design.
2. Enhance Your website functionality.

and much more ideas, you can use it.

== Installation ==

1. Upload `WPMK Block` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Simply download and upload / Installed and active the plugin.
